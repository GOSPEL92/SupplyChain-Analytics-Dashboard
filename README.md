# Supply Chain Analytics Dashboard (Power BI)

This dashboard consolidates SKU profitability, supplier quality, inspection results, cost efficiency, and stock replacement analysis.

## Key Features
- KPI banner: Revenue, Profitability, Defect Rate, % SKUs Passed Inspection
- Supplier quality analysis to reduce defect costs
- Inventory optimization to minimize stockouts
- AI Copilot integration for automated reporting
